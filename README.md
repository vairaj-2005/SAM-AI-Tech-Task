# AWS S3 Basic Cloud Storage Setup

## Objective
To create a cloud storage solution using Amazon S3, upload files, manage access permissions, and retrieve stored objects.

## Services Used
- Amazon S3 (Simple Storage Service)
- AWS Management Console

## Steps Performed
1. Created an S3 bucket in AWS.
2. Uploaded a sample image file to the bucket.
3. Configured access permissions for the object.
4. Generated the object URL.
5. Verified successful retrieval of the uploaded file through the URL.

## Result
Successfully created and configured an Amazon S3 bucket, uploaded files, managed permissions, and accessed the stored object using its public URL.
